## データの変更を監視

----

1. watch:{変数名:funciton(){}}
 変数を監視して、変更時に処理を行う
 配列やオブジジェクトの中身まで捕捉してほしい時は、以下のように
 deep:trueをつける。

```javascript
export default {
    watch:{
        data:{
            handler:function(val) {
                this.renderChart(this.data, this.options);
            },
            deep:true
        }
    }
}
```
